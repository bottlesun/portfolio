# ZoomCloneCoding

## [ ZoomClone App ]


API를 활용하여 뉴스 페이지 웹 사이트 구현하기

* Demo App - https://bottlesun.github.io/study/04-ZoomCloneCoding/

## ●기능
* SOCKETIO & WEBSOCKETS 이용 하여 로컬 백엔드 서버 구축
* 방 번호 입력시 해당 방으로 이동 하는 화면 구현
* 실시간 채팅기능 구현
* 캠 화면 및 마이크 연결 실시간 라이브 기능
* 캠 위치 변경 기능
* 반응형 홈페이지 구현

## ●Skill
* Pug
* CSS
* JavaScript
* SOCKETIO & WEBSOCKETS
